# Heritage Treasures: An In-Depth Analysis of UNESCO World Heritage Sites in Tableau
"Heritage Treasures: An In-Depth Analysis of UNESCO World Heritage Sites (2019)" is a comprehensive project aimed at exploring the rich dataset of UNESCO World Heritage Sites using Tableau. This project focuses on visualizing the distribution, trends, and key attributes of these sites to provide valuable insights.By leveraging the power of data visualization, stakeholders can gain a deeper understanding of the global heritage landscape, identify patterns, and make informed decisions to enhance the preservation and promotion of these sites.

Dataset Link: https://www.kaggle.com/datasets/ujwalkandi/unesco-world-heritage-sites/data?select=whc-sites-2019.csv
